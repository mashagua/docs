# DesireCore v10.0.64

**Release Date**: 2026-05-19

![v10.0.64 更新概览](/img/changelog/v10.0.64-en.png)

## New Features
- **Enhanced PDF Reading**: The Read tool now supports reading large PDF files up to the GB level, and automatically selects a more suitable loading method based on file size, reducing issues where large files fail to open or consume too much memory.

## Fixes
- The Read tool can now convert scanned or image-based PDFs into page-by-page images for visual model recognition, avoiding the previous issue where these types of PDFs would return empty content after being read.
- Compatibility issues when exporting Skills, Agents, Teams, and related data as ZIP files have been fixed, reducing export failures.

## Improvements
- When reading PDFs, the Read tool now focuses more on the first page by default and limits the number of pages read in a single pass, reducing context usage and processing costs while maintaining effectiveness.
- The built-in default resource pack has been synced to the latest version, so new installations or initializations can get updated default configurations and content.
