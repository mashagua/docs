# DesireCore v10.0.67

**Release Date**: 2026-05-20

![v10.0.67 更新概览](/img/changelog/v10.0.67-en.png)

## Fixes
- For long-running agent tasks, when API retries occur, previously generated tool calls, reasoning steps, and conversation fragments will no longer be lost, allowing task records to be preserved more completely.

## Improvements
- For tasks that were affected in the past, a recovery solution is now available to reconstruct missing message records from session events, helping restore previously lost tool call history.
