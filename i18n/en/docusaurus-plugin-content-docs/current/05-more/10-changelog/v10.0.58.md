# DesireCore v10.0.58

**Release Date**: 2026-05-14

![v10.0.58 更新概览](/img/changelog/v10.0.58-en.png)

## Fixes
- Fixed an issue on Windows where automatic updates could sometimes fail with “Unable to uninstall old version files.” Users installed in either a local directory or Program Files can now upgrade more reliably.
- Fixed an issue in merged conversation history where repeatedly editing the same user message could leave stale content behind. After editing, only the latest version will be retained.
- Fixed an issue where old topic memory could still carry over into subsequent conversations after chat history was truncated. Context in follow-up replies will now be more accurate.
