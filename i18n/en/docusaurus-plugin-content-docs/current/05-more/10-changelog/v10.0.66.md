# DesireCore v10.0.66

**Release Date**: 2026-05-19

![v10.0.66 更新概览](/img/changelog/v10.0.66-en.png)

## Fixes
- Fixed an issue in asynchronous delegation scenarios where messages, delegation cards, and sub-agent reports could appear in the wrong order. The order shown during real-time conversation now remains consistent after refresh.
- Fixed an issue where content written back asynchronously by sub-agents could be sent to the wrong conversation. Delegation results now reliably return to the current conversation context.
