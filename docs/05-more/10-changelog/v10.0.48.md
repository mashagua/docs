# DesireCore v10.0.48

**发布日期**: 2026-05-08

![v10.0.48 更新概览](/img/changelog/v10.0.48.png)

## 新功能
- **心跳通知文件链接**：智能体在心跳通知中可附带生成的文件，您可以直接从通知卡片或历史记录中点击打开，Markdown 文件支持用 SuperDoc 查看，其他文件会用系统默认方式打开。
- **多智能体并发心跳**：现在多个智能体的定时心跳任务可以更稳定地同时运行，彼此不会互相占用或干扰。

## 修复
- Workflow 在创建或校验时会更早拦截运行时并不支持的图结构问题，避免出现“校验通过但执行结果异常”的情况。
- Workflow 的分支预演与校验提示更加明确，能更清楚地看到条件分支、回退路径和引用变量，减少排查配置问题的成本。

## 改进
- 心跳通知逻辑更贴近实际结果：有结果时更积极提醒，确认没有变化时才保持静默，减少无效打扰。
- 心跳记录现在会保留通知摘要和相关文件信息，方便您后续回看每次心跳产生了什么结果。
- 心跳系统对工具调用、超时控制和文本内容清洗做了整体优化，使定时任务执行更稳定、通知内容更清晰。
- Workflow DSL 规范与提示进一步统一，编写工作流时对字段写法、分支约束和人工审批参数的要求会更清楚。
