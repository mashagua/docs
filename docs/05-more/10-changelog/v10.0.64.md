# DesireCore v10.0.64

**发布日期**: 2026-05-19

![v10.0.64 更新概览](/img/changelog/v10.0.64.png)

## 新功能
- **PDF 阅读增强**：Read 工具现在支持读取 GB 级大型 PDF，并会根据文件大小自动选择更合适的加载方式，减少大文件无法打开或内存占用过高的问题。

## 修复
- Read 工具现在可以把扫描版或图片型 PDF 按页转成图片交给视觉模型识别，避免此前这类 PDF 读取后内容为空的问题。
- 导出技能、Agent、团队及相关数据为 ZIP 时的兼容性问题已修复，导出失败的情况更少了。

## 改进
- Read 工具读取 PDF 时默认更聚焦于首页，并限制单次读取页数，在保证效果的同时减少上下文占用和处理成本。
- 内置默认资源包已同步到最新版本，新安装或初始化时可获得更新的默认配置与内容。
