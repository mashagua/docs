# DesireCore v10.0.58

**发布日期**: 2026-05-14

![v10.0.58 更新概览](/img/changelog/v10.0.58.png)

## 修复
- 修复了 Windows 上自动更新时可能出现“无法卸载旧版本文件”的问题，安装在本地目录或 Program Files 的用户现在都能更稳定地完成升级。
- 修复了合并后的对话历史中反复编辑同一条用户消息时旧内容残留的问题，编辑后只会保留最新版本。
- 修复了截断聊天历史后旧话题记忆仍可能被带入后续对话的问题，后续回复的上下文会更准确。
